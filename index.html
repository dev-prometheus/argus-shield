Protect Your Wallet From Drainer Attacks

Argus Shield is an open-source browser extension that detects and blocks malicious signature requests before they reach your wallet.

Download Extension Installation Guide

Advanced Threat Detection

Real-time analysis of every signature request and transaction to identify known attack vectors.

Permit2 Protection

Detects PermitBatchTransferFrom and PermitTransferFrom signatures that can drain multiple tokens with a single click.

EIP-2612 Detection

Identifies native token permit signatures that work on fresh wallets without any prior approval transactions.

Approval Analysis

Warns when approving tokens to Permit2 or unknown addresses, preventing the setup for future signature-based attacks.

Blind Sign Block

Automatically blocks dangerous eth_sign requests that can sign arbitrary data without showing contents.

Whitelist System

Pre-configured with trusted DeFi protocols. Add your own verified addresses to reduce unnecessary warnings.

Activity Monitoring

Tracks all intercepted requests with detailed logs. Know exactly what threats were blocked on each site.

How Argus Works

Seamless protection that runs in the background without affecting your Web3 experience.

1

Request Interception

Argus hooks into your wallet provider and intercepts all signature and transaction requests before they reach your wallet.

2

Threat Analysis

Each request is analyzed in real-time to detect Permit2 signatures, native permits, dangerous approvals, and known attack patterns.

3

Risk Assessment

Requests are classified as Critical, High, or Medium risk based on the potential impact to your assets.

4

User Decision

If a threat is detected, you see a detailed warning modal with the option to block the request or proceed if you trust the source.

Installation Guide

Get protected in under 2 minutes. No account required.

1

Download the Extension

Download the latest release ZIP file from GitHub Releases

2

Extract the ZIP

Unzip the downloaded file to a folder on your computer (e.g., argus-extension)

3

Open Extensions Page

In Chrome, Brave, or Edge, navigate to chrome://extensions

4

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page

5

Load the Extension

Click "Load unpacked" and select the argus-extension folder you extracted

6

Verify Installation

You should see the Argus Shield icon in your toolbar. Click it to verify protection is active.
